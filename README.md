<h3 align="center">movie bingo</h3>

  <p align="center">
    A fun side project.  Sit down and watch a movie (currently only supports scary movies), play this bingo game and see who wins Bingo or Blackout first.  When users play the game they will be able to select the movie that they are watching and the game will log when that event occurred.  When more people play the game it will learn about more movies.  Eventtually it will be able to know when an event should happen when watching a movie.  It can then notify the user if they didn't click it, or the game could make an even chance for everyone (future features).  There is an option to select or create a room for your friends or family to use also.

## Getting Started

To get a copy running:
1. clone the repo -   
   ```sh
   git clone https://github.com/mbolsen/MovieBingo.git
   ```
2. get the necessary libraries -   
   ```sh
   npm install npm@latest -g
   ```
3. get your local mongodb running
4. start the server -
   ```sh 
   npm start
   ```
